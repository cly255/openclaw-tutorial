📖 什么是 OpenClaw？

  • 🌐 搜索网络信息

⚙️ 前置准备

在安装 OpenClaw 之前，你需要确保电脑上已经安装了以下软件：

1. Node.js（版本 18 或更高）

OpenClaw 基于 Node.js 运行，如果没有安装，请访问：

https://nodejs.org 下载 LTS 版本

1

验证安装：

node --version

如果显示版本号（如 v18.x.x 或更高），说明安装成功。

2. Git

用于克隆 OpenClaw 源码：

https://git-scm.com

2

验证安装：

git --version
✅ 完成检查： 确保以上两个软件都已安装并可用。 如果遇到问题，先解决它们再继续下一步。

💻 Windows 系统安装

🐚 Nugget 提示： 如果你是 Windows 用户，可能需要以管理员身份运行 PowerShell！

步骤 1：打开命令行

按 Win + X，选择 "Windows PowerShell（管理员）" 或 "终端（管理员）"

步骤 2：克隆 OpenClaw 仓库

git clone https://github.com/openclaw/openclaw.git

这会在当前目录创建一个 openclaw 文件夹。

步骤 3：进入项目目录

cd openclaw

步骤 4：安装依赖

npm install

这一步会下载所有需要的包，可能需要几分钟，请耐心等待 🍵

步骤 5：配置环境

复制配置文件模板：

copy .env.example .env

然后编辑 .env 文件，填入你的 API Key（如 OpenAI、StepFun 等）。

步骤 6：启动 OpenClaw

npm start

看到类似以下输出说明成功了：

🦞 OpenClaw 已启动！
📊 状态: 运行中
🔗 网关: http://localhost:12128
🎉 祝使用愉快！
✅ 恭喜！ 你的 OpenClaw 已经在 Windows 上跑起来了！

📱 接入飞书教程

让 OpenClaw 通过飞书机器人接收和发送消息！

准备工作

  • 一个飞书企业账号（需要有应用管理权限）
  • OpenClaw 已经在运行（访问 http://localhost:12128）
  • 准备好 API Key 和 Secret
⚠️ 注意： 以下步骤假设你使用的是飞书机器人（自定义机器人）， 而不是企业自建应用。如果你要集成飞书消息 API，需要额外配置。

步骤 1：在飞书创建机器人

  1. 打开 飞书开发者平台
  2. 进入 "应用" → "创建应用"
  3. 选择 "机器人" 类型
  4. 填写应用名称、描述
  5. 上传机器人头像（可以用我们的 🦞 龙虾图标！）

步骤 2：配置机器人权限

  1. 在 "权限管理" 页面，添加以下权限：
    • 获取用户信息
    • 发送消息
    • 接收消息
  2. 保存并发布应用

步骤 3：获取 Webhook 地址

  1. 在机器人设置页面，找到 "自定义机器人"
  2. 开启 "接收消息" 功能
  3. 复制 Webhook URL（以 https://open.feishu.cn/ 开头）

步骤 4：在 OpenClaw 配置飞书集成

  1. 登录 OpenClaw 管理后台
  2. 进入 "集成" → "飞书"
  3. 点击 "添加机器人"
  4. 填入：
    • Webhook URL: 上一步复制的地址
    • 签名密钥: 飞书后台提供的签名 Key
    • 机器人名称: 比如 "Lobster 助手"
  5. 点击 "测试连接"
  6. 测试成功后在飞书群里 @机器人，应该能收到回复！
🎯

高级配置（可选）

  • 消息过滤 - 设置机器人只响应 @ 自己的消息
  • 自动回复 - 配置关键词自动回复
  • 权限控制 - 限制哪些群/用户可以对话
✅ 搞定！ 现在你的 OpenClaw 已经在飞书上上岗工作了！ 试着在群里 @它，看看它怎么回应吧～ 🦞💬

❓ 常见问题

Q1: npm install 失败怎么办？

A： 常见原因和解决方案：
  • 网络问题：使用国内镜像 npm config set registry https://registry.npmmirror.com
  • 权限问题：Windows 用管理员权限，Mac/Linux 加 sudo
  • Node 版本过低：升级到 Node 18+

Q2: 端口 12128 被占用怎么办？

A： 修改 .env 文件中的 PORT 为其他端口（如 3000）， 或者关闭占用端口的程序：netstat -ano | findstr :12128 查找并结束进程。

Q3: 飞书机器人收不到消息？

A： 检查以下几点：
  • Webhook URL 是否填写正确
  • 签名密钥是否匹配
  • 机器人是否已发布并添加到群组
  • OpenClaw 是否正常运行（检查日志）

Q4: 如何升级 OpenClaw？

git pull origin main && npm install

然后重启 OpenClaw 即可。

Q5: 可以同时接入多个平台吗？

A： 当然可以！OpenClaw 支持微信、飞书、Telegram、Discord 等多平台同时接入。 在 "集成" 页面逐一配置即可。

Q6: 我的 API Key 安全吗？

A： API Key 只存储在本地 .env 文件中，不会上传到任何服务器。 请妥善保管，不要分享给他人。建议使用有使用限制的 Key。

Q7: 如何自定义 AI 回复风格？

A： 在 "技能" → "系统提示词" 中修改 System Prompt， 或者创建自定义技能来调整 AI 的行为。

Q8: OpenClaw 支持哪些 AI 模型？

A： 支持 OpenAI GPT 系列、Claude、StepFun（阶跃星辰）、文心一言、通义千问等主流模型。 你需要自己提供 API Key。
